 60 | ||| (Infinitary) coproduct of monoids
 61 | ||| It is a subtype of the Pi type with finite support
 62 | ||| This means that in the finite case it coincides with the monoid product
 63 | ||| But in the infinite case we need to be explicit about the support:
 64 | ||| which elements of the domain map to nonzero, and what are their values?
 65 | ||| This ends up being equivalent to the graph of the function of all the 
 66 | ||| nonzero elements, i.e. a list of input-output pairs such that
 67 | ||| 1) order of the list does not matter (quotiented out by permutations)
 68 | ||| 2) IO pairs where the output is zero is same as not having them, i.e.
 69 | |||    `(i, 0) : xs = xs`
 70 | ||| 3) input duplicates add the outputs, i.e if the same input `i` appears twice
 71 | |||    then  `(i, xi) : (i, xj) : xs = (i, xi + xj) : xs`
 72 | ||| All maps out of this type have to satisfy these constraints
