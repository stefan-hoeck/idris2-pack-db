 12 | This file defines functions which perform pure optimisation:
 13 | optimisation of a differentiable function `f : p -> x` (here modelled as a lens `f : p =%> x` via some optimiser such as gradient descent. 
 14 | Here no "loss function" or "input-output pairs" are needed, just a function to optimise.
 15 |
 16 | This file provides functionality for creating, and turning supervised learning problems into pure optimisation problems, via a function which takes:
 17 | a) a parametric lens `f : x >< p =%>> y`
 18 | b) a loss function `loss : (y, y) =%> l`
 19 | c) input-output pairs `IO (x.Shp, y.Shp)`
 20 | and composes them to produce an optimisation problem `f : p =%> l` which the above described functions can consume.
 21 |
 22 | Notably, only a *non-dependent* supervised-learning problem can be turned into a pure optimisation one. If the parameter space depends on the input, then learning becomes its own thing.

 44 | ||| Performs function optimisation for `numSteps : Nat` times, while logging 
 45 | ||| the progress to the console.
 46 | public export
 47 | optimise : {p, x : AddCont} -> InterfaceOnPositions x Num =>
 48 |   {default 100 printEvery : Nat} ->
 49 |   {default Nothing initParam : Maybe p.Shp} ->
 50 |   Show p.Shp => Show x.Shp => Show stateTy =>
 51 |   (fIO : IO (MAddLens {m=IO} p x)) ->
 52 |   (opt : Optimiser p stateTy) ->
 53 |   (numSteps : Nat) ->
 54 |   IO (p.Shp, stateTy)
 55 | optimise fIO opt numSteps = do
 56 |   currentValue <- case initParam of
 57 |     Just p => pure p
 58 |     Nothing => opt.initParam
 59 |   currentState <- opt.initState
 60 |   runActionUntilMaxSteps
 61 |     {l=x.Shp}
 62 |     {printEvery=printEvery}
 63 |     (optimiseStep fIO opt)
 64 |     numSteps 0
 65 |     (currentValue, currentState)
 66 |     (\pSt => do
 67 |       loss <- fIO
 68 |       loss.fwd (opt.fwd pSt))
