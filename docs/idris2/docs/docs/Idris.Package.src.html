 407 |     --
 408 |     -- We use a backtracking algorithm here: If several versions of
 409 |     -- a package are installed, we must try all, which are are
 410 |     -- potentially in bounds, because their set of dependencies
 411 |     -- might be different across versions and not all of them

 414 |         List Depends ->
