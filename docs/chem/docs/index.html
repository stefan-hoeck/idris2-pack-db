  • Chem.QSAR.HAcceptor
    Functionality for counting the number of hydrogen bond acceptors in a molecule. Like in the CDK, a H-acceptor is either * a non-positively charge oxygen that is not an aromatic ether nor adjacent to a nitrogen * a non-positively charge nitrogen that is not adjacent to an oxygen
  • Chem.QSAR.HDonor
    Functionality for counting the number of hydrogen bond donors in a molecule. Like in the CDK, a H-donor is either * a non-negatively charge oxygen bound to at least on hydrogen * a non-negatively charge nitrogen bound to at least on hydrogen
  • Chem.QSAR.JPlogP
