data BoundedQueue : Nat -> Nat -> Type -> Type

  track of its size, with amortized O(1) enqueue and dequeue operations.
  The `m` argument tracks the `BoundedQueue`s limit,
  and the `n` argument tracks the `BoundedQueue`s size.
