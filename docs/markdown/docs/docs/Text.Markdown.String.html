  A horizontal rule.
  
  The horizontal rule is represented by three or more consecutive '*', '-', or
  '_' characters.
  
  A blank line should be placed before and after the horizontal rule, unless
  at the end of the beginning or end of the text.
  
  ```markdown
  Hello
  
  ---
  
  World
  ```
