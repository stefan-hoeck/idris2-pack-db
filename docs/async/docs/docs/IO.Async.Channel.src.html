 87 | ||| A concurrent, bounded channel holding values of type `a`.
 88 | |||
 89 | ||| This is an important primitive for implementing single
 90 | ||| consumer, multiple producer services.
 91 | |||
 92 | ||| Note: Unlike with `IO.Async.BQueue`, which can have multiple
 93 | |||       consumers, this will only accpet a single consumer,
 94 | |||       silently overwriting an old consumer in case a new one
 95 | |||       calls.
