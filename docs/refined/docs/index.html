    This module derives functionality for refinement type. In general, a refinement type is of the following shape: ```idris record Percent where constructor MkPercent value : Double 0 prf : IsPercent value ``` That is, a value paired with a predicate on that value. The predicate can be an explicit or implicit argument of any quantity.
